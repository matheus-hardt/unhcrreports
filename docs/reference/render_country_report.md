# Render Country Report

Renders a Quarto report for a specific country and year.

## Usage

``` r
render_country_report(country_iso3, year, output_dir = ".", output_file = NULL)
```

## Arguments

- country_iso3:

  ISO3 code of the country (e.g., "COL").

- year:

  Numeric year.

- output_dir:

  Directory to save the report.

- output_file:

  Filename for the report.
