# Summary Key Figures

Key figures can be highlighted with humanitarian icons
https://fontawesome.com/icons/categories/humanitarian

## Usage

``` r
plot_ctr_keyfig(year = 2021, country_asylum_iso3c = country_asylum_iso3c)
```

## Arguments

- year:

  Numeric value of the year (for instance 2020)

- country_asylum_iso3c:

  Character value with the ISO-3 character code of the Country of Asylum

## Value

ggplot2

## Examples

``` r
plot_ctr_keyfig(
  year = 2022,
  country_asylum_iso3c = "COL"
)
```
